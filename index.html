Summary

Rusimg is a image processing CLI tool written by Rust.

It can be used to process images in various ways, such as resizing, trimming, compressing, converting formats, and more.

Futhermore, Rusimg can be used as a library, so you can use it in your Rust project.

Features
Converting
Supports: BMP, JPEG, PNG, WebP

Converts the format of the image.

Supports conversion to BMP, JPEG, PNG, and WebP.

And more, Rusimg library can be added new formats as extension.
